Detailed Description

Interface for compressed bit vectors that use pasta's rank and select interface.

This interface is currently work in progress and should not yet be used in production.

pasta::BlockCompressedBitVector::BlockCompressedBitVector ( BitVector &&  bv)
inline

Constructor for a block compressed bit vector that takes a BitVector and prepares the compression.

To easier work with the rank and select data structures, the bit vector is only compressed as soon as compress() is called. If a rank and select data structure is computed for this compressed bit vector, this happens automatically, e.g.,

BitVector bv(1000);
// Fill bv
BlockCompressedBitVector bcbv(bv);
FlatRankSelect(std::move(bcbv));
